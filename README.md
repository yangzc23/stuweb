### 项目介绍
使用到的框架主要包括：**JDK + servlet + sqljdbc4**

数据库：**SQL Server**

服务器软件：**tomcat**

开发工具：**eclipse + maven**

功能点如下：

**1）添加学生（已完成）**

**2）学生列表（已完成）**

3）列表分页显示（未完成）

**4）学生详情（已完成）**

**5）修改学生（已完成）**

**6）删除学生（已完成）**

7）搜索学生（未完成）

8）后台出现异常界面能够给出简单清晰的错误提示（未完成）

9）表单校验的JS功能（未完成）

### 思路
1）HttpServlet的作用：对http请求进行处理（service方法）

2）HttpServletRequest表示请求（获取请求里面参数的内容）

3）HttpServletResponse表示响应（发送响应数据给浏览器）
